# 15. 명령 패턴
: 유사한 동작을 하나의 객체로 묶어 실행하는 행위 패턴

### 명령 처리
- 작업 동작 자체를 전달하려 처리
### 명령 패턴의 구성과 특징
- 인터페이스
- 명령
- 리시버
- 인보커

### 명령패턴 == 트랜잭션 패턴
#### 매개변수
- 명령 패턴은 객체의 실행 동작 시점을 분리하여 지연 시키는데, 절차지향의 콜백 함수와 유사
#### 시점 제어
- 작업의 요청과 처리를 분리, 큐에 명령을 쌓아두고 실행
#### 복구
- 명령 취소 처리도 추가할 수 있다.
#### 저장
- 명령 동작을 임시 저장하여 재실행 가능
