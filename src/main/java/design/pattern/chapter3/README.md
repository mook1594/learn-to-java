- 팩토리 메서드 패턴, 템플릿 메서드 패턴은 유사하다
- 두 패던의 공통점: 추상화를 사용해 객체 생성, 상위클래스에서 정의를 결정하고 하위 클래스에 구체적인 처리를 위임
